## Run Urlkv

1. Start redis server in terminal

```
$ redis-server
```

2. Run Urlkv.py

```
python3 Urlkv.py
```

The regex expression for URL Validation is from https://github.com/django/django/blob/stable/1.3.x/django/core/validators.py#L45
